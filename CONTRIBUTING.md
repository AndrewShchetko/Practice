## Setting Up the project locally
Make sure that you have pipenv
1. Clone repository
    ```shell
    git clone git@github.com:AndrewShchetko/Practice.git
    ```
2. Install dependency
    ```shell
   cd Practice/
    pipenv install   
   ```
3. Run virtual environment
   ```shell
    pipenv shell
   ```
